Description

Prestige est une entreprise spécialisée dans les champagnes de luxe pour les hôtels chinois. L'objectif de l'entreprise est de permettre aux hôtels de commander des champagnes personnalisés pour leurs clients.

CompétencesSuite AdobeMarketingDate2020

Maquette de l'application

L'application créé comporte plusieurs étapes. Premièrement, le client (responsable de l'hôtel) choisis son type de champagne. Puis, il le personnalise en fonction de ses envies (forme de la bouteille, taux de bulles, texte sur la bouteille, emballage...). Une fois cette étape réalisée, le client peut alors réserver un rendez-vous où il sera accueilli là où le champagne est produit. Sur les lieux, il pourra rencontrer le personnel, tester le champagne, visiter les environs, etc. Enfin, pour la dernière étape, il choisit le nombre de bouteille qu'il souhaite faire réaliser.

Description de l'application

Plusieurs étapes ont été nécessaire afin de parvenir à la création de l'application. Il a fallu décider des différentes options que propose l'application, une roadmap, la simulation du parcours client, une analyse du marché chinois, des études de la culture du pays afin de bien choisir les couleurs, les termes employés, etc.
